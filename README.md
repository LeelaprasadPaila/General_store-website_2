# 🛒 General Store Website

A simple and responsive **General Store Website** built using **HTML and CSS**. This project was created as my very first step into web development, where I learned the fundamentals of structuring webpages and designing user interfaces.

## 🌐 Live Demo

**View the project here:**
https://leelaprasadpaila.github.io/General_store-website_2/

---

# 📖 About the Project

This website is a basic front-end design for a general store. It includes a clean layout with sections that represent products, categories, and basic navigation.

The main purpose of this project was to understand:

* HTML page structure
* CSS styling
* Layout design
* Navigation bars
* Images and sections
* Responsive design basics
* Building and deploying a website using GitHub Pages

* 🛍️ Simple General Store homepage

Every developer has a first project—this one is mine.
